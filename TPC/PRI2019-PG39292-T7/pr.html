    • Abstract

    O objetivo deste trabalho de casa foi criar um servidor web em node js, utilizando a framework Express com uma base de dados que recolhemos através do MONGODB, criando assim uma API de dados. Tivemos que converter o ficheiro das obras de XML para JSON para de seguida importarmos para o MONGO. De seguida tivemos que utilizar vários pedidos GET para o servidor mostrar as obras, compositores e uma lista de obras do compositor. /obras /obras?periodo=X /obras?anoCriacao=Y obras?compositor=A /compositores /compositores/:nome
