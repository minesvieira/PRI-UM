    • Abstract

    O objetivo deste trabalho de casa foi criar um servidor web em node js, utilizando a framework Express com uma base de dados que recolhemos através do MONGODB. Tivemos de utilizar vários tipos de pedidos: GET (consultas), POST (envio de informação) e DELETE (pedido para apagar informação) Este servidor tem uma página inicial em que clicamos num botão para ver a listagem de filmes com o ID, titulo e data. Nessa interface temos outro botão para criar um novo filme, em que vamos ter um pequeno formulário com 4 campos para adicionar o titulo, data, atores e géneros. Na listagem de filmes podemos ver mais informações sobre esse filme, ou seja, quando clicamos mostra-nos o id, data, nome, géneros e atores sobre esse filme. Por fim, temos também a opção de eliminar um filme.
