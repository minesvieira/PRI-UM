    • Resumo

    No primeiro exercício, tive que passar o dataset de xml para json para o conseguir utilizar, importando-o para o mongo. Este ficheiro encontra-se na pasta ex1 chamado de musicas.json Para executar esta aplicação coloquei o servidor do mongoDB e criei a base de dados chamada "musicas". Utilizei o seguinte comando, na diretoria do exercicio 1 para a criação dessa bd: mongoimport -d musicas -c musicas --jsonArray C:\Users\Maria\Desktop\Teste_recurso\ex1\musicas.json Depois disto, corri o servidor com npm start dentro da pasta do exercicio 1 e comecei a construir a api de dados para as obras. No segundo exercício, comecei por responder às perguntas usando o Postman e também coloquei diretamente o link da api pretendida no navegador para obter as respostas. Essas respostas encontra-se na pasta respostaPostman dentro da pasta ex2, com o nome de questoes.txt. De seguida, comecei a desenvolver a aplicação Nodejs. Para executar a aplicação também desenvolvida no segundo exercício, basta executar o servidor dentro da pasta ex2 com npm start. Para o exercicio 1 usei a porta 3000 e para o segundo exercicio a porta 3002.
